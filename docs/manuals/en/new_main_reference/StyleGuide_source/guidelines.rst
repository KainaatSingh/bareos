.. _Guidelines:

Guidelines for Writing the Bareos Documentation
===============================================

.. _General:

Headings
--------

Kindly use the following punctuation characters for the particular headings:

   1. Use ``=`` for Chapters:

      .. code-block:: sh
      		
		Chapter
      		=======

   2. Use ``-`` for Sections: 
      
      .. code-block:: sh
      
		Section     
      		-------

   3. Use ``~`` for Subsections:

      .. code-block:: sh
      
		Subsection
      		~~~~~~~~~~
